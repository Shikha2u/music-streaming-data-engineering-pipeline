# Sample Data (Author-Created)

These CSV files contain **synthetic sample data created for this project**. They populate a fictional music streaming database with artists, songs, listeners, subscriptions, and interaction records.

| File | Description |
|------|-------------|
| `artist.csv` | Artist profiles (name, country, genre, bio) |
| `song.csv` | Song catalog (title, duration, genre, release date) |
| `listener.csv` | Listener accounts (demographics, subscription) |
| `listener_account_type.csv` | Subscription tiers (Free, Individual, Family, etc.) |
| `song_artist.csv` | Song-to-artist relationships |
| `song_interaction.csv` | Plays, likes, shares, skips |
| `follow_artist.csv` | Listener follow relationships |

This data is **not** sourced from any commercial streaming service.

See [docs/04-synthetic-dataset.md](../docs/04-synthetic-dataset.md) for full documentation.
